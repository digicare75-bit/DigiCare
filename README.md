# 🏥 DigiCare

DigiCare is a modern healthcare and online pharmacy web application designed to provide users with seamless digital healthcare services.

## 🚀 Features

- 🔐 User Authentication (Signup/Login)
- 📧 Email OTP Verification
- 💊 Online Medicine Ordering
- 🏥 Healthcare Services Access
- 📱 Responsive Design
- 🛒 Cart & Checkout System

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
